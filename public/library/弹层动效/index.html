主要内容

注意：目前该页面的几个弹层不能在同一个页面同时使用；

打开demo2后，刷新再打开demo3；

demo2:点击按钮打开弹层，点击x关闭弹层

注意：目前适用pc和手机，但不兼容ie8

使用方法：

第一步：

给需要使用该风格弹层的内容一个class为container的类名

第二步：

$('.demo2').click(function() {

var message = '标题';

var title = '内容';

tancengDemo2.showTanCeng(message,title);

});

demo3:点击按钮触发弹层后，鼠标滑上页面出现弹层内容，滑出消失弹层

注意：目前只适用pc，但不兼容ie8，也不适用手机

使用方法：

$('.demo3').click(function() {

var message = '标题';

var title = '内容';

tancengDemo3.showTanCeng(message,title);

});

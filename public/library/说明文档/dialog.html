说明文档

当前是duoduo的个人分支，谁都别动!!! 注意事项：

1. 插件文件不要太多，js/css/html文件最好各不要超过2个，保持功能简单单一

2. 上传的时候只选择文件上传（文件夹是传不上去的），后台会自动生成文件 夹

3. 所有文件都统一在一个目录下，不要针对CSS/JS建文件夹，没有必要，而且服务端目前也不支持

4. demo页面请写明插件使用方法和注释, 并标明作者

5. 上传文件必须包含index.json和demo的html页面, index.json是配置文件，参数如下

          {
          "index": "index.html", //demo页面名字
          "name":"test11", //可以忽略
          "imgUrl":"../img/product1.jpg" //目前可以忽略
        }

6. index.json文件里面不能有任何注释!!!

7. 未完待续。。。

点击我要上传按钮，弹出区域，直接点击或者拖动文件到该区域即可，项目名称必填
